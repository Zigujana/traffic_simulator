📋 How to Use This Simulation (Zimbabwe Traffic Standards)

  • Traffic Light System: Red = Stop, Amber = Stop if safe (Zimbabwe rule), Green = Go
  • All-Red Phase: Both directions red briefly for intersection clearance (safety standard)
  • Safe Following Distance: Cars maintain 2-second rule + minimum gap automatically
  • Collision Prevention: Advanced detection prevents cars from crashing
  • Crush Prediction: Cars detect potential collisions and brake preemptively
  • Speed Adjustment: Cars automatically adjust speed based on traffic ahead
  • Visual Indicators: Green zones show safe following distance, brake lights activate when stopping
  • Sensitivity Analysis: Adjust one parameter at a time to observe traffic flow changes

🚦 Zimbabwe Traffic Light Sequence:

  • Phase 1: Direction A: Green → Amber → Red
  • Phase 2: All Red (clearance phase)
  • Phase 3: Direction B: Green → Amber → Red
  • Phase 4: All Red (clearance phase) → Repeat
